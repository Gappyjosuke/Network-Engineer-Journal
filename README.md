# UngaBhai Technologies: NetOps Challenge Academy

Welcome to **UngaBhai Technologies**, a rapidly growing enterprise headquartered in Chennai. This repository is a real-world network engineering training ground. Instead of blindly configuring routers, you will step into the shoes of a **Junior Network Engineer** facing architectural problems, design reviews, and real-world failure states.

This isn't an exam simulator; it is a exercise in developing **engineering judgment**.

---

##  How It Works

1. **Pick an Operation:** Browse the `Operations/` directory. Each folder contains a real business scenario with specific constraints.
2. **Design First, CLI Second:** Open the `DESIGN_TEMPLATE.md` within the operation folder. Write out your design architecture before touching Packet Tracer or real hardware.
3. **Submit Your Design:** Fork this repo, add your solution to the `Solutions/` folder, and open a Pull Request to challenge the community review.

---

##  The Engineering Core Values

Every operation is evaluated against 5 critical production metrics, not raw test percentages:
*   **Planning:** Did you map your dependencies and constraints out before typing?
*   **Documentation:** Can a midnight engineer read your notes and save the data center?
*   **Naming Standards:** Is your syntax structured across global branches?
*   **Scalability:** Will this break when the company doubles in size next quarter?
*   **Cleanliness:** Are your unused interfaces safely shutdown, isolated, and documented?
