# CCNA Notes

> [!NOTE]
> This repository contains my personal CCNA study notes while preparing for the Cisco Certified Network Associate (CCNA). The notes are written for my own understanding.
>
> This repository previously contained Packet Tracer labs, networking projects, and other practical exercises. Those materials have been archived and are **no longer available** in this repository.
>
> The repository has since been repurposed into a dedicated base focused on networking concepts, protocol internals, Cisco IOS, and structured CCNA notes.
